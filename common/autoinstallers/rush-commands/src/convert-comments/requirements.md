# 代碼倉庫中的中文備註轉換爲英文

## 簡介

一段 JS 腳本，用於將代碼倉庫內的中文備註轉換爲英文。

## 執行邏輯：

1. 調用 git 命令，先查出本倉庫下所有源碼文件
2. 讀取文件內容，分析文件中(可能包含 go 和ts、js、md 等代碼，主要關註文本型文件)是否包含中文備註，是的話將文件目錄加進待處理數組 tasks 中
3. 遍歷 tasks 中的文件，提取出中文代碼備註，調用 openai 接口，進行翻譯，翻譯後插入源位置
4. 所有文件均處理完畢後，輸出報告

## 命令行參數：

命令名稱：ai-translate
參數：

- `root`: 需要執行命令的目錄
- `exts`: 需要進行處理的文件擴展名，支持數組，默認爲空

## 技術約束

- 使用 ts 編寫代碼
- 調用 openai api 做翻譯
- 使用 commander 實現命令行交互
- 優先使用 FP 編寫代碼
