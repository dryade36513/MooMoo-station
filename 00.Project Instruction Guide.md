# 00.Project Instruction Guide

簽名：CodeCraft Sensei
日期：2026-02-22

## 專案資料夾掃描結果

| 資料夾 | 推測功能模組 |
|---|---|
| `.claude/` | Claude AI 相關設定與代理設定 |
| `.github/` | GitHub Action 工作流程、腳本與 Issue 模板 |
| `.vscode/` | VSCode 開發環境設定 |
| `backend/` | 後端核心代碼庫 |
| `backend/api/` | 對外 API 介面、中介軟體 (Middleware) 與路由設定 |
| `backend/application/` | 應用層邏輯 (App, 連接器, 對話, 知識庫, 記憶, 權限, 工作流等) |
| `backend/bizpkg/` | 業務相關的共用套件 (設定, LLM, 檔案工具等) |
| `backend/conf/` | 後端設定檔 |
| `backend/crossdomain/` | 跨領域/模組的業務邏輯 |
| `backend/domain/` | 領域驅動設計 (DDD) 的領域層核心模型與邏輯 |
| `backend/infra/` | 基礎設施層 (資料庫 ORM, 快取, 儲存, 向量資料庫等) |
| `backend/pkg/` | 通用基礎套件 (錯誤處理, I18n, 執行緒安全等) |
| `common/` | 前後端共用資源或腳本 |
| `docker/` | Docker 容器化部署設定與資料庫初始化 (Atlas, Nginx 等) |
| `docs/` | 專案說明文件 |
| `frontend/` | 前端核心代碼庫 (Monorepo 架構) |
| `frontend/apps/coze-studio/` | 主前端應用程式 |
| `frontend/packages/` | 前端內部共用套件庫 (架構, 元件, 狀態管理, 工作流等) |
| `helm/` | Kubernetes Helm Charts 部署設定 |
| `idl/` | 介面定義語言 (Thrift)，用於定義 RPC 介面和資料結構 |
| `scripts/` | 開發、建置、部署用的輔助 Shell 腳本與 Go 腳本 |
