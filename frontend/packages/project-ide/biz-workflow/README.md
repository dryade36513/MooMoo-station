# @coze-project-ide/biz-workflow

A workflow package for the MooMoo Station monorepo

## Overview

This package is part of the MooMoo Station monorepo and provides workflow functionality. It serves as a core component in the Coze ecosystem.

## Getting Started

### Installation

Add this package to your `package.json`:

```json
{
  "dependencies": {
    "@coze-project-ide/biz-workflow": "workspace:*"
  }
}
```

Then run:

```bash
rush update
```
